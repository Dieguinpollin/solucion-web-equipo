Portal de Eventos Estudiantiles - FCC

Problemática


Es común que los alumnos olviden fechas clave solo porque no hay un solo lugar donde revisar todo lo que pasa en la facultad. 


Objetivo


Crear una web sencilla con un calendario de fechas, una lista de próximos eventos y un formulario para que los alumnos se registren en actividades.


Roles del equipo


Project Manager (Diego): Gestión del repo, control de "Issues" y aprobación de los Pull Requests del equipo .


Desarrollador Front 1 (Gael): Creación del calendario

Desarrollador Front 2 (Taylor): Creación del formulario de registro y el pie de página (footer) .



Conflicto en el repositorio 


Para cumplir con la resolución del conflicto en equipo, realizamos lo siguiente:


Qué lo causó: El equipo de GitHub no había configurado correctamente el editor de texto en el entorno de desarrollo.


Cómo lo arreglamos?: Nos juntamos a revisar el código, borramos manualmente las líneas raras que pone Git (<<<<, ====) y dejamos las dos mejoras juntas para que el texto quedara bien. Al final, guardamos y subimos el archivo ya corregido.


Qué hicimos: Nos encontramos con un error en el editor de texto que no permitía la edición de texto en el entorno de desarrollo.
