Portal de Eventos: "Al Día"

Problemática

Es común que los alumnos olviden fechas clave solo porque no hay un solo lugar donde revisar todo lo que pasa en la facultad. 

Objetivo

Crear una web sencilla con un calendario de fechas, una lista de próximos eventos y un formulario para que los alumnos se registren en actividades.
